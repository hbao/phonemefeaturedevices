The XML API for Java(tm) ME (JSR 280) provides Java ME subsets of several XML APIs; the specification is composed jointly of these javadocs and the specification document (javame-xml-spec.pdf).

The XML APIs in the JSR 280 specification include subsets of:

  • Java API for XML Processing (JAXP)
  • Simple API for XML Processing (SAX)
  • Streaming API for XML (StAX, JSR 173)
  • Document Object Model (DOM) Core
  • DOM Events and View
